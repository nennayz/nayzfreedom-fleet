# Agent Team Brief for Graphic Design

## Summary
นี่คือสรุปทีม Agent ของ NayzFreedom / NayzFreedom Fleet เพื่อใช้ส่งต่อให้ฝ่ายกราฟิกออกแบบภาพทีมงานและสร้าง Visual Identity ของแต่ละคน

ทีมนี้เป็น multi-agent pipeline สำหรับการผลิตคอนเทนต์โซเชียลมีเดีย ตั้งแต่การค้นหาเทรนด์จนถึงการเผยแพร่และชุมชน

## Design Brief for Graphic Team

### 1. Robin — Chief of Staff
- Persona: เป็นผู้บริหารฉลาด สุภาพ มีวิสัยทัศน์แบบ strategist
- Character traits: organized, decisive, calm, confident, professional
- Visual direction: ชุดลายทางหรือชุดสูทสไตล์โมเดิร์น, โทนสีกรมท่า / เทา / ขาว
- Props: แท็บเล็ต, โน้ตบุ๊ก, แผนภูมิ, ไอคอนเครื่องหมายถูก
- Emotional tone: dependable, authoritative, guiding

### 2. Mia Trend — Trend Researcher
- Persona: ขยัน, สังเกตดี, analytic, เป็นนักสำรวจข้อมูลแบบทันสมัย
- Character traits: curious, energetic, sharp, nimble
- Visual direction: ชุด casual ที่มีลายไดนามิก, โทนสีเขียวมะนาว / ฟ้า / ขาว
- Props: แว่นตา, มือถือ/คอมพิวเตอร์, กราฟเทรนด์, แว่นขยาย
- Emotional tone: alert, inquisitive, slightly playful

### 3. Zoe Spark — Idea Generator
- Persona: ครีเอทีฟ, มีพลัง, คิดนอกกรอบ และชวนตื่นเต้น
- Character traits: imaginative, optimistic, fast, bright
- Visual direction: สีสันสดใส เช่น ส้ม / เหลือง / ชมพู, เสื้อผ้าที่ดูมี movement
- Props: หลอดไฟ, ไอคอน flash, ฟองความคิด, กระดาษสเกตช์
- Emotional tone: fun, inspiring, motivating

### 4. Bella Quill — Script Writer
- Persona: มีสไตล์, ละเอียด, วรรณศิลป์, เข้าใจเสียงแบรนด์
- Character traits: articulate, persuasive, elegant, empathetic
- Visual direction: ใช้โทนอบอุ่น เช่น เบจ / น้ำตาลอ่อน / ทอง, ชุดที่ดูคลาสสิก
- Props: ปากกาพู่กัน, โน้ต, เรื่องสคริปต์, ลายมือเรียบหรู
- Emotional tone: smooth, professional, story-driven

### 5. Lila Lens — Visual Director
- Persona: สายสร้างสรรค์, มองเห็นภาพรวม, รู้จัก composition และ mood
- Character traits: stylish, visionary, technical, artistic
- Visual direction: โทนม่วง / น้ำเงิน / เทา, เครื่องแต่งกายดู arty และทันสมัย
- Props: กล้อง, tablet, moodboard, paint splashes, camera lens icon
- Emotional tone: creative, thoughtful, polished

### 6. Nora Sharp — QA Editor
- Persona: รอบคอบ, ตาเหยี่ยว, พร้อมให้ feedback ที่เป็นประโยชน์
- Character traits: precise, critical, honest, supportive
- Visual direction: ชุดที่ดูเป็นผู้เชี่ยวชาญ เช่น สีเทาเข้ม / น้ำเงิน, แว่นตา
- Props: สัญลักษณ์ checklist, แว่นขยาย, กระดาษ feedback, ปากกา
- Emotional tone: fair, exacting, reassuring

### 7. Roxy Rise — Growth Strategist
- Persona: เชี่ยวชาญการเติบโต, เข้าใจแพลตฟอร์ม, ชัดเจนในผลลัพธ์
- Character traits: strategic, upbeat, data-driven, actionable
- Visual direction: โทนสีส้ม / เขียวเข้ม / ดำ, ชุดสปอร์ตสุดโก้
- Props: แฮชแท็ก, นาฬิกา, กราฟ, กรอบเวลาโพสต์
- Emotional tone: forward-looking, energetic, informed

### 8. Emma Heart — Community Specialist
- Persona: อ่อนโยน, เป็นมิตร, ฟังดี, โปร์แอคทีฟ
- Character traits: warm, empathetic, clear, caring
- Visual direction: โทนสีชมพูอ่อน / ม่วง / ขาว, ชุดสบายๆ หรือ corporate casual
- Props: ฟองข้อความ, ไอคอนหัวใจ, แชทบ็อกซ์, โน้ตตอบคำถาม
- Emotional tone: trustworthy, welcoming, engaging

## Visual Style Notes
- ให้ทีมดึงคอนเซ็ปต์ทีม AI เป็น “ทีมงานมืออาชีพที่ทันสมัย”
- ควรมีความรู้สึกว่าแต่ละ Agent มีบทบาทเฉพาะ และทำงานร่วมกันเป็น ecosystem
- แนะนำให้ใช้กราฟิกสไตล์ semi-flat หรือ modern illustration ที่มีเส้นสะอาด
- สร้าง palette หลัก 3-4 สี เพื่อให้ภาพทีม cohesive แต่ยังแยกเฉพาะตัวเองได้
- อาจเพิ่ม element ร่วม เช่น เส้นเชื่อม, วงกลมปฏิบัติการ, ไอคอนเทคโนโลยี

---

## Brief for the Graphic Designer
1. **วัตถุประสงค์**: สร้างทีม AI agent character set สำหรับใช้งานบนหน้าเว็บไซต์, presentation, pitch deck หรือ internal brand kit
2. **ต้องการ**: ภาพทีม (group shot) และภาพเดี่ยวของแต่ละ Agent อย่างน้อย 8 คน
3. **ต้องมี**:
   - ชื่อ Agent
   - ตำแหน่ง / บทบาท
   - Persona สั้น ๆ
   - สีหลักของตัวละคร
   - ไอคอนหรือ props ที่สื่อความหมาย
4. **แนวทาง tone**:
   - ประกอบด้วยความเท่, ฉลาด, พร้อมรับมือธุรกิจ digital marketing
   - พื้นผิวสะอาด ไม่หวือหวาเกินไป
   - แบ่งสีตาม Agent แต่ยังรู้สึกเป็นทีมเดียวกัน

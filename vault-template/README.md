# NayzFreedom Vault Template

This is a starter structure for a private Obsidian vault.

## Important

This folder is a **template**, not the recommended place for private life notes.

Recommended actual private vault location:

```text
~/Documents/NayzFreedom Vault/
```

Copy this folder when creating the real vault. The real `vault/` folder is ignored by git.

## What belongs here

- Captain profile
- Freedom Five notes
- ship overviews
- mission reflections
- song ideas
- travel memories
- decisions and personal reviews

## What does not belong in a committed template

- API keys
- passwords
- private financial details
- health records
- private relationship notes
- anything you would not want committed to git
